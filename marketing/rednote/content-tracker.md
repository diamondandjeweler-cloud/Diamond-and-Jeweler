# DNJ 小红书 · 发布流程 + 数据追踪

每天发完填一行。每周日看一次。数据决定下一步做什么。

> 看不懂指标先看 `README.md` §5、§9。匹配口径红线见 `README.md` 顶部。

---

## A. 每条笔记发布流程（照着走）

1. **查选题** → `content-calendar-120day.md` 找当天（形式 + 标题 + 支柱）。
2. **取文案** → `caption-bank-day1-60.md` / `day61-120.md` 复制（标题 + 封面 + 正文 + 话题）。
3. **做封面** → 3:4，藏青/白，大字钩子（规则见 `README.md` §6）。标 `*` 的先换真实素材。
4. **做内容** → 图文：套模板出图；视频：竖屏 9:16 + 字幕 + 卡点音乐。
5. **发布** → 标题 ≤20 字、正文末尾放话题、定位大马（可选）、按 §发布时间。
6. **黄金 1 小时** → 秒回评论 + 去 8–10 条同类热门笔记真诚留言。
7. **记录** → 回填下方表格。

**发布时间（MYT）：** 视频 12:00–13:00 或 20:00–22:30；图文 12:30 / 21:00；互动帖 19:00–20:00。
