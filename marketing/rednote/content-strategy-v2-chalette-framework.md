## 三、改造 #2 — 内容 4 步对照

### ① 选题（Topic）— 加「30 秒爆款扒款」周流程

**Chalette 教法：** 搜客户搜的词 → 切「最热」 → 抓赞藏量高的笔记 → 拆解结构 → 套用.

**DNJ 当前：** 我们有「搜索词」（写在每篇 caption 里），但没有**系统化扒爆款**.

**新增 SOP（每周日 30 分钟）：**

| 步骤 | 动作 | 工具 |
| :-: | :-- | :-- |
| 1 | 搜 5 个核心词（马来西亚求职 / 面试技巧 / 简历 / 应届生 / 谈薪） | RedNote 搜索栏 |
| 2 | 切「最热」 | 顶部 filter |
| 3 | 截图 top 10 笔记（封面 + 标题 + 赞数） | 手机截屏 |
| 4 | 写入 `content/hot-research/YYYY-MM-DD.md`（标题模板、封面元素、钩子句） | Markdown |
| 5 | 下周 2-3 篇内容套用结构（不抄文案，抄"结构 + 钩子句式"） | — |

### ③ 标题/图文 — DNJ 服务类公式

**Chalette 公式（虚拟服务）：** `共鸣 + 方法感 + 解决方案`

**DNJ 标题对照：**
- "应届生必看｜投100份0回复，错在这3点" → 共鸣✅ + 方法感(3点)✅ + 解决(暗示)✅ ⭐⭐⭐⭐⭐
- "钻石不会偶然遇见珠宝匠" → 共鸣✅ + 方法感❌ + 解决❌ ⭐⭐
- "5个真的有用的面试回答" → 方法感✅ + 解决✅ + 共鸣弱 ⭐⭐⭐⭐
- "5步谈薪法，第4步最多人漏" → 方法感✅ + 解决✅ + 共鸣弱 ⭐⭐⭐⭐

**改造规则：** 每个标题至少要有 2/3 个元素（共鸣 + 方法感 + 解决方案）.

### ④ 封面 — 4 大元素

**Chalette 标准：**
- 字大（一眼看清）✅ DNJ 大字大体合格
- 颜色鲜艳（红/黄/绿高对比）⚠️ DNJ 全用 navy + gold — 品牌一致但**视觉刺激低**
- 3:4 比例（推荐流最佳）⚠️ 需 audit
- 带数字 ✅ 大部分有（100份/6秒/3点/5步）

**改造 A/B 测试计划：**

为 qday19 / qday101 / day45 做 **2 个版本**：
- 版本 A（现有 navy/gold）
- 版本 B（高对比：红橙底 + 白字 + gold 钻石 logo 角落）

**A/B 评估方式：** 同主题不同时段发布或下个月做对照测试.

**封面公式三选一（DNJ 是虚拟服务）：**
```
共鸣（POV 句 / 戳痛点句）
  + 方法感（数字 + 步骤）
  + 解决方案（结果暗示）
```

**例（Day 45 谈薪 7 图）：**
- 当前封面：「5 步谈薪法 ｜ 第 4 步最多人漏」(方法感 + 共鸣✅)
- 优化版：「谈薪｜以为开口要钱？错！」（戳痛点共鸣 → 主图变第 2 张：5 步公式）
