## 一、去哪里找互动对象（按话题搜 + 蹲热门笔记）
**话题（在小红书搜索框逐个搜，进「最热」笔记）：**
- `#马来西亚求职` `#大马打工人` `#大马招聘` `#吉隆坡找工作` `#槟城找工作`
- `#应届生求职` `#毕业生找工作` `#留学生求职` `#海归求职`
- `#面试` `#面试技巧` `#简历` `#谈薪` `#转行` `#裸辞` `#职场`
- `#求职焦虑` `#海投` `#已读不回` `#空窗期`

**账号类型（关注 + 持续互动）：**
- 大马求职 / 职场博主、HR / 猎头号、应届生 / 留学生分享号、职业规划 / 简历辅导号
- 在马的留学生社群、毕业生 KOC（中腰部，1k–5w 粉，互动率高，适合合拍/连麦）

## 二、每日互动 SOP（CES 加权：关注+8 ＞ 评论/转发+4 ＞ 收藏/赞+1）
> 新号阶段控量，避免风控（参考：每天关注 ≤20–30、评论 ≤20–30，节奏放慢）。

1. **蹲 5–10 条同类热门笔记**（求职痛点 / 干货），在评论区**真诚留言给价值**（不是打广告）：
   - 干货帖：补一条实用建议（如「自我介绍可以用『现在→过去→未来』90秒公式」）
   - 情绪帖：共情 + 鼓励（如「海投到自我怀疑那段太真实了，抱抱」）
2. **关注**这些笔记的作者 + 评论区里「正在求职」的活人（精准潜在用户）。
3. **回流自己评论区**：发布后黄金 1 小时秒回每条评论，把高质量问答置顶。
4. **引导**：在合适处自然带「关注我，下一个被看见的就是你」；不硬塞链接。

## 三、每日配额（冷启动期，安全节奏）
| 动作 | 每日上限 | 说明 |
|--|--|--|
| 关注 | 20–30 | 优先评论区里的真实求职者 |
| 评论 | 15–25 | 真诚、相关、给价值；不复制粘贴同一句 |
| 点赞 | 30–50 | 配合浏览自然进行 |
| 收藏 | 自然 | 收藏同行优质干货，学习选题 |

## 五、合作 / 裂变（中期）
- 和大马本地中腰部**求职/职场/留学**博主合拍、连麦、互推。
- 发起「30 天求职自律打卡」「免费简历诊断」征集，让粉丝替你「代言」，带来精准信任流量。

- ❌ 评论里发广告 / 链接 / 刷屏同一句 → 会被限流甚至封号。
- ❌ 玄学词；❌ 广告法极限词（最/第一/绝对/万能）。
