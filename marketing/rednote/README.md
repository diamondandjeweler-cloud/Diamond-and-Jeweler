运营 **DNJ 小红书号** 的唯一总指挥文件。这是 Instagram「Hidden Talent」引擎的中文小红书版。
**目标：** 90 天 0 → 1 万粉。**引擎：** 日更 + 每周 ≥3 条视频 + 干货收藏 + 情绪转发 + 评论区运营。

> ⚠️⚠️ **第一红线 · 全程死守：** 匹配机制对外**只能**说「**AI 智能匹配 / 数据精准匹配**」。
> **绝不**出现：八字、命理、生辰、五行、风水、算命、星座、缘分、命中注定、玄学、命盘——任何一个都不行。
> 收集出生日期，只说「AI 匹配所需资料」，**不解释用途**。若有人评论区问「你们是不是看命 / 算的」，
> 标准回复：「我们用的是一套综合多维度信号的 AI 匹配模型」——不承认、不否认、不展开。

## 4. 每天的固定动作

每天就做这 5 件事：
1. **发当天主笔记** — 在 `content-calendar-120day.md` 查当天选题，去对应 `caption-bank` 复制文案，配好封面（见 §6）。
2. **黄金 1 小时** — 发布后蹲守，**秒回每一条评论**（小红书评论互动权重极高）。
3. **主动出击** — 去 8–10 条同类热门笔记下，留**真诚、有价值**的评论（不是打广告）。
4. **维护私信** — 有人问就认真回，引导到主页 / 建档。
5. **记录** — 在 `content-tracker.md` 填一行（曝光、点赞、**收藏**、**评论**、涨粉）。

## 5. 增长引擎（卡住了就回来看）

- **封面 = 命门。** 小红书 80% 点击靠封面。每条都要：大字标题钩子 + 干净藏青/白底 + 一眼看懂利益点。
- **收藏 > 点赞。** 干货合集驱动**收藏**，收藏把你推给更多人。每周保证 2–3 条强干货合集。
- **转发靠情绪。** 「被已读不回」「终于收到 offer」这类情绪帖最容易被转发给朋友 = 涨粉第一杠杆。
- **视频带涨粉。** 每周 ≥3 条视频笔记，钩子前 3 秒定生死，**永远加字幕**，结尾落「主页链接 / diamondandjeweler.com」。
- **标题公式：** 数字 + 痛点 + 情绪 emoji，≤20 字。例：「投了73份简历，0回复😭」。
- **配比：** 35% 情绪共鸣 · 25% 干货 · 20% 被低估的天赋 · 12% 招聘真相 · 8% 产品/职位。
- **爆了怎么办：** 单条曝光破 1 万 → 下周换钩子重发（Day 112 就是干这个）+ 考虑投「薯条」RM5–10。
- **不涨了怎么办：** 找大马本地中腰部博主（求职 / 应届 / 生活类）合拍 / 连麦；加大情绪帖比例。

**里程碑（记录在 `content-tracker.md`）：** 第2周 → 200 · 第4周 → 600 · 第6周 → 1,800 · 第9周 → 4,000 · 第13周 → 8,000 · Day 90 → 10,000。

## 6. 封面制作规则（最重要的一节）

每条笔记都要做封面。统一风格 = 账号辨识度。

- **尺寸：** 竖版 3:4（1080×1440）最吃流量。
- **配色：** 藏青 `#1E2A5E` 底 + 白字，或 白底 + 藏青字。**不用金色、不放真钻石图。**
- **字体：** 标题用超粗黑体（思源黑体 Heavy、站酷高端黑、阿里巴巴普惠体 Bold）。
- **排版：** 大字标题占封面 60% 以上，1 句钩子，最多再加 1 行小字。左上 / 底部放小 logo「DNJ 💎」。
- **每条封面文案**已在 `caption-bank` 里给好（「**封面文案**」那一行）。
- **可复用 IG 资产：** `marketing/instagram/content/` 里的语录卡 / 合集图可改中文重用；生成器是 `make_quote_cards.py`、`make_carousels.py`（改文字 → 重跑）。

**形式对应：**
- **图文合集 / 语录卡** → 用 Canva / 醒图 / 稿定 套模板，3:4，首图即封面。
- **视频笔记** → 竖屏 9:16，剪映加字幕 + 卡点音乐，封面单独做一张 3:4 大字图。

## 9. 每周复盘（每周日，从 `content-tracker.md` 看）

1. 收藏最高的是哪条？→ 复制选题，再做 2 条同类。
2. 转发 / 评论最高的情绪帖？→ 复用那个钩子。
3. 完播率（过 3 秒）最好的视频开场？→ 沿用结构。
4. 最差的一条？→ 砍掉那个角度。
5. 净增粉 vs 上周 → 是否在奔向下个里程碑。
6. 有没有破 1 万曝光的？→ 换钩子重发 + 考虑投薯条。
