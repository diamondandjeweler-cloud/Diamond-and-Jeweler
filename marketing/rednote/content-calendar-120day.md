# DNJ 小红书 · 120 天内容日历（目标：90 天破 1 万粉）

**账号：** DNJ·被低估的天赋　·　**定位：** 大马华人求职 / 招聘　·　**人设声音：** 被低估的天赋运动（情绪 > 企业感）

> 这是 Instagram「Hidden Talent」内容引擎的小红书中文版。同一套品牌内核，按小红书的玩法重做：**封面为王、标题抓人、干货收藏、情绪转发**。
>
> ⚠️ **保密红线（全程严格执行）：** 匹配机制对外一律说「**AI 智能匹配 / 数据精准匹配**」。**绝不**出现 八字 / 命理 / 生辰 / 五行 / 风水 / 算命 / 星座 / 缘分 / 命中注定 等任何玄学或命理词。收集出生日期只说「AI 匹配所需资料」，不解释用途。

## 增长引擎（发之前先看）

- **封面决定生死。** 小红书 80% 的点击来自封面。每条都要有大字标题钩子 + 干净背景（藏青 `#1E2A5E` / 白底）。
- **收藏 = 推荐。** 干货合集带来收藏，收藏把你推给更多人。情绪共鸣带来转发，转发带来新粉。两手都要。
- **标题公式：** 数字 + 痛点 + 情绪 emoji。20 字以内。例：「投了73份简历，0回复😭」。
- **发布后黄金1小时：** 秒回每一条评论；去 5–10 条同类热门笔记下真诚留言（不是打广告，是给价值）。
- **内容支柱配比：** 35% 情绪共鸣 · 25% 实用干货 · 20% 被低估的天赋 · 12% 招聘真相 · 8% 产品/职位。
- **形式：** 视频笔记带来涨粉爆发；图文/合集建立收藏与信任。每周至少 3 条视频。
- **卡住了就加速：** 和大马本地中腰部博主合拍 / 连麦；对最好的一条投「薯条」RM5–10。

**四个阶段：**
| 阶段 | 天数 | 目标 | 主打 |
|--|--|--|--|
| 一 · 冷启动 | 1–30 | 立人设、攒第一批粉 | 打工人共鸣 + 运动宣言 |
| 二 · 涨粉爆款 | 31–60 | 收藏 + 转发起量 | 干货合集 + 情绪高点 |
| 三 · 信任深化 | 61–90 | 双边信任、软植入 | 故事 + 雇主视角 + 真相 |
| 四 · 转化社区 | 91–120 | 案例转化、建社区 | 真实案例 + 职位 + 宣言 |
